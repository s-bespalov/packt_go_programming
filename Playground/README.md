# BFS ASCII Playground

Учебный проект демонстрирует поиск пути на сетке 10×15 (150 узлов) алгоритмом BFS, учитывающем препятствия, и визуализирует все ключевые этапы поиска в ASCII-графике.

## Запуск

```sh
cd Playground
go run .
```

Программа показывает первые несколько шагов поиска и финальный кадр с найденным маршрутом. На каждом шаге отображаются: очередь (ограничена до 8 позиций, дальше `…`), список посещённых узлов (ограничен в 20 элементов) и ASCII-сетка, где:

- `S` — старт (`00,00`)
- `G` — цель (`09,14`)
- `#` — препятствие
- `@` — текущая вершина обработки
- `o` — узлы в очереди (фронтир)
- `.` — уже посещённые узлы
- `*` — финальный путь (если найден)

Сетка отрисована в виде таблицы с подписями строк и столбцов, к тому же в этом примере задан набор препятствий вручную.
